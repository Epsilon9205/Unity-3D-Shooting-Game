## Overview
Unity 6 と C#で制作した3Dシューティングゲームです。

## 操作方法
- WASD：移動
- 左クリック：弾を発射
- Space：ゲーム開始
- R：リスタート

## 実装機能
- タイトル画面
- プレイヤー移動
- 敵の自動スポーン
- 敵の追尾AI
- 弾の発射・当たり判定
- スコア・HP表示
- ボス戦
- ボスのホーミング弾
- GAME OVER / GAME CLEAR
- 爆発エフェクト
- 3Dモデル・Skybox

## 工夫した点
ボス戦、ホーミング弾、爆発エフェクト、Prefabを用いた敵生成などを実装し、ゲームとして遊べる流れを作りました。
